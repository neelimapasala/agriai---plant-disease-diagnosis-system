## 🌟 Overview

**Neelima's AgriAI** is a revolutionary AI-powered plant disease diagnosis system designed specifically for farmers. It provides instant disease detection, treatment recommendations, and agricultural advisory services in **6 Indian languages**.

## 🚀 Features

### Core Features
- 🔬 **AI Disease Diagnosis** - Upload leaf images for instant disease detection
- 📸 **Camera Capture** - Take photos directly from the app
- 📝 **Text Description** - Describe symptoms in natural language
- 🎯 **95% Accuracy** - Advanced AI algorithms for precise diagnosis

### Innovative Features
- 🌐 **6 Indian Languages** - English, Hindi, Telugu, Tamil, Kannada, Marathi
- 🔗 **Blockchain Security** - Tamper-proof diagnosis records
- 🌤️ **Weather Integration** - Real-time disease risk prediction
- 💬 **AI Chatbot** - 24/7 farmer assistance
- 📊 **Market Prices** - Live crop prices with trends
- 👨‍🌾 **Expert Consultation** - Book video/phone consultations
- 🏛️ **Government Schemes** - Subsidy and insurance information
- 🌱 **Soil Health Analyzer** - NPK levels and recommendations
- 🐛 **Pest Identification** - Guide for common pests
- 📅 **Crop Calendar** - Planting and harvest schedules
- 💰 **Yield Predictor** - Profit and yield estimation
- 📱 **QR Reports** - Share diagnosis with experts
